# Implementation Plan

Create a detailed implementation plan for: $ARGUMENTS

## Steps

1. **Analyze Requirements**
   - Understand the task scope
   - Identify dependencies
   - Consider constraints

2. **Break Down Tasks**
   - Create smaller, manageable tasks
   - Define acceptance criteria
   - Estimate effort

3. **Identify Risks**
   - Technical risks
   - Dependencies
   - Timeline risks

4. **Create Timeline**
   - Define milestones
   - Set deadlines
   - Plan checkpoints

5. **Document Plan**
   - Write clear documentation
   - Include all decisions
   - Reference related work

## Output

Provide a structured implementation plan with:
- Overview
- Tasks with acceptance criteria
- Risks and mitigation
- Timeline
