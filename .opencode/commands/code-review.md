# Code Review

## Steps

1. **Understand Context**
   - Read related issues/PRs
   - Understand the purpose

2. **Review Code**

3. **Identify Issues**

   - Minor suggestions

4. **Provide Feedback**

   - Suggest solutions
   - Prioritize issues

- Strengths
- Issues (categorized)
- Suggestions
- Approval status
